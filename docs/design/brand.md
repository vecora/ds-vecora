## Sizes

| Class            | Height | Width |
| ---------------- | ------ | ------|
| `logo--nano`     |   24px |  15px |
| `logo--tiny`     |   32px |  20px |
| `logo--small`    |   48px |  30px |
| _no class_       |   64px |  40px |
| `logo--large`    |   96px |  60px |
| `logo--big`      |  128px |  80px |
| `logo--huge`     |  256px | 160px |
| `logo--massive`  |  512px | 320px |

Pixels sizes assumes 1 [unit](docs/introduction/#eight-point-grid) = 8px.
